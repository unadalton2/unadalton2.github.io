Mancala

This is a simple cheat designed for Game Pigeon’s Mancala (Avalanche Mode). Originally programmed in Java, this project has been ported over to javascript for the demo below. This Mancala cheat works by fully exploring all possible moves a player can make in a single turn then sorting them by the total scores they result in. Usually it is able to win the game within one or two turns.
