FaceAI

FaceAI is a simple autoencoder capable of manipulating faces. Using tensorflow and keras for the creation and training, FaceAI is able to compress images from their raw rgb format to 12.5% the original size. FaceAI does this by maping single values within the compressed image to larger more general shapes and/or colors in the final image. The high overfitted compression allows the smooth interpolation between faces. Meaning shapes and/or colors smoothly change positions and form.



Training

FaceAI was trained using tensorflow and keras. The model was trained for about 300 epochs using 809 yearbook photos as training data. The lack of data almost certainly means the model is overfitting. A couple of solutions to this would be getting more data or adding noise to existing data. Overfitting gives the model a lot of power to explore the existing dataset allowing the model to reach higher compression rates before hitting a local minimum.



Iterpolation

Interpolation is the smooth blending between two points. In this case we are blending faces together. The model has two main methods to interpolate between faces. The first fading, this method layeres one picture on top of the other then fades the overlaid image. The model doesn't use a ton of fading unless it's trying to interpolate between very distant faces or when switching between genders as hair takes up a lot of space. The second method is much more interesting, morphing between faces. Morphing means altering the location and size of facial features making one face look more like another. Morphing's main benefit is the ability to create a realistic face throughout interpolating between faces. Unfortunately morphing is almost never used alone, instead FaceAI uses a mix of fading and morphing to interpolate between faces.
